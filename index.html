Automated Door Controller

About our Project...

This Project uses a Passive Infrared (PIR) motion sensor and a servo motor to create a simple motion-activated system. The PIR sensor is connected to digital pin 7, and the servo motor is connected to digital pin 8. The program constantly reads the PIR sensor's output. If motion is detected (HIGH signal), the servo motor moves to a 180-degree position, simulating an action like opening a door, and then waits for 3 seconds before resetting. If no motion is detected (LOW signal), the servo motor moves to a -200-degree position, representing a closed or initial state. The Serial monitor is used to display the sensor's output for debugging purposes. This project creates a basic motion-triggered servo system, which could be expanded for various applications such as automated doors or interactive projects.
